Ferramentas

Vamos utilizar o Visual Studio Code , esse programa e uma ferramenta open Soure, mas existem outros. Para desenvolver uma página web podemos utilizar qualquer editor de texto, como bloco de notas por exemplo.

Após baixar e instalar o VScode vamos instalar a extensão LiveServe, que será muito útil para leitura dos html como pagina no navegador. Outra ferramenta/extensão no Vscode é a do próprio Html, mas essa extensão já vem instalado no programa. Emmet, é uma ferramenta muito usada, ela cria uma página html já pré-definida.

Inspeção dos elementos é um console/ferramenta muito utilizado para estudar como a pagina web foi criada, com ele conseguimos ver toda a estrutura do html que quisermos. Para acessarmos ele basta clicar F12 no teclado e o console será aberto.

Estrutura básica do Html

Primeiro vamos criar uma pasta específica de trabalho, depois vamos abri-la no Vscode. Criamos um arquivo index.html (index significa pagina principal). O Html consiste em 3 partes, são elas;

< html > - que é toda a pagina;

< head > - tudo que estiver entre ele são as coisas que irão carregar antes de exibir para o usuário;

< body > - tudo que estiver entre ele são as coisas que irão aparecer para o usuário que acessar na página.

As Tags < >

As tags dão uma função, tudo o que estiver entre elas o navegador irá interpretar de uma forma.

< > - usamos as tags para aplicar a função

< i > - Deixa o texto em itálico

< strong > - Deixa o texto em negrito

< imput > - essa tag não tem fechamento, usa-se ela para criar campo de digitação

< img / > coloca uma imagem

Os atributos das tags

São propriedades que uma tag pode ter, existem as globais que todas as tags irão ter, e outras específicas de cada tag

< strong id="titulo" > o id é utilizado como identificador de elemento

< strong style="color:blue" > o style e usado para alterar a cor do texto

< strong class="titulo-principal" > usado para toda a seleção ficar com a mesma cor.

Textos

Para criação dos textos são usadas algumas tags que servirão para formatação edição e marcações.

< h1 > usado para títulos

< h2 > para subtítulos

< p > para o texto, são os parágrafos

< blockquote > usado para citação

< u > cria underline nas palavras

< mark > marca texto

Listas

Existem os esquemas de listas também, que são as listas ordenadas < ol > e as listas não ordenadas < ul >.

Para o uso das < ol > devemos usar as tags < li > que são suas tags filhas para identificar o padrão da ordem das listas.

ex: < ol >

< li > Item 1< /li >

< li > Item 2< /li >

< /ol >

Já para < ul > não é necessário, as lista não ordenadas criarão marcadores. Essa tag e muito utilizada para criar menus com a utilização do CSS.

Link

Usamos < a > de ancor, ancorar, para ligar ou ligar os links das página/html..

< a href="site" > o nome < /a >

< a href="site" target="_blank" > texto < /a > - esse atributo é usado para a paginar abrir em outra aba.

< hr > cria uma ina abaixo do texto.

< a href="#local do id x" > esse atributo ira fazer a movimentação até o id que for definido.

Ex: < h2 id=x" >
